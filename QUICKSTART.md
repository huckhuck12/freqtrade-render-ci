## 方式一：独立测试 (5分钟) 🆕

**最简单的开始方式，无需任何复杂配置！**

### 步骤1：准备环境
```bash
# 克隆项目
git clone <your-repo-url>
cd freqtrade-render-ci

# 安装基础依赖（通常系统已有）
pip install pandas numpy
```

### 步骤2：运行测试
```bash
python scripts/local/test_strategy.py
```

### 步骤3：查看结果
你将看到类似这样的输出：
```
=== 最终策略回测结果 ===
初始资金: $10,000.00
最终资金: $14,712.00
总收益率: 47.12%
胜率: 58.10%
```

**🎉 恭喜！你已经成功运行了策略测试！**

## 方式二：本地回测 (15分钟) 🔧

**使用真实数据进行更准确的回测**

### 步骤1：安装Freqtrade
```bash
pip install freqtrade
```

### 步骤2：快速回测
```bash
# 使用默认参数（推荐）
./scripts/local/run_local_backtest.sh

# 或自定义参数
./scripts/local/run_local_backtest.sh 20240101-20241231 "ETH/USDT BTC/USDT" 365
```

### 步骤3：等待结果
脚本会自动：
1. 下载历史数据
2. 运行回测
3. 显示结果摘要

## 方式三：GitHub Actions (自动化) 🤖

**适合团队协作和持续集成**

### 步骤1：Fork仓库
点击GitHub页面右上角的"Fork"按钮

### 步骤2：触发回测
**自动触发**：
- 创建Pull Request
- 推送到main分支

**手动触发**：
1. 进入Actions页面
2. 选择"Manual Backtest - 8PM Strategy"
3. 点击"Run workflow"
4. 设置参数并运行

### 步骤3：查看结果
- PR中会自动显示回测结果
- Actions页面可下载详细报告

## 🆘 遇到问题？

### 常见问题快速解决

**Q: 提示"freqtrade未安装"**
```bash
pip install freqtrade
```

**Q: 权限错误**
```bash
chmod +x scripts/local/*.sh
```

**Q: 网络问题导致数据下载失败**
- 检查网络连接
- 尝试使用VPN
- 减少数据天数参数

**Q: 想修改策略参数**
- 编辑 `scripts/local/final_optimized_strategy.py`（独立测试）
- 编辑 `user_data/strategies/EightPMHighLowStrategy.py`（Freqtrade）
